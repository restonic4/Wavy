# üìù Commit Message Convention

To keep our history clean and readable, please use the following symbol-based prefixes for your commit messages:

| Symbol | Action | Description |
| :--- | :--- | :--- |
| `[+]` | **Add** | New features or functionality |
| `[-]` | **Fix** | Bug fixes or minor changes |
| `[!]` | **Breaking** | Breaking changes or a critical change |
| `[#]` | **Meta** | Documentation, comments, etc |

**Example:** `[+] Add dark mode support`
