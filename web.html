Authentication Endpoints 
POST /auth/register - Register new user
POST /auth/login - Login (sets cookie)
POST /auth/logout - Logout (clears cookie)
GET /auth/me - Get current user

All use credentials: 'include' for cookies
User Management 
POST /users/{id} - Update user (self or admin)
DELETE /users/{id} - Delete user (self or admin)

Users can modify their own account
Admins can modify any account
Song Management (Admin Only) 
GET /songs - List all songs
POST /songs - Create song metadata
POST /songs/upload - Upload audio file
GET /songs/{id} - Get song details
POST /songs/{id} - Update song
DELETE /songs/{id} - Delete song

Admin cookie required for POST/DELETE
Streaming & Listeners 
GET /stream - MP3 audio stream
POST /api/heartbeat - Send listener status
GET /api/status - Get listeners + history

Cookie optional (anonymous if no cookie)
